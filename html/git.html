Git

  • git clone ссылка с GitHub - Склонируем удаленный репозиторий с GitHub себе на компьютер.
  • git status - просмотреть изменения произошедшие в папке.
  • git add - добавить файл в отслеживаемые файлы.
  • git commit -m — сделаем коммит. Так мы покажем Git, что файл изменен. Изменения файла сохранятся в репозитории
  • git push - внесем изменения на удаленный репозиторий.
  • git pull — изменения подтянуться из удаленного репозитория в локальный.
